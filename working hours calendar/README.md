# 28時間チェックカレンダー

アルバイトの勤務時間を記録し、7日間の合計勤務時間を確認できるカレンダーです。

## Features

* 年月を `YYYYMM` 形式で入力してカレンダーを表示
* 前月・次月への移動
* 日ごとの勤務時間を入力
* 月間の合計勤務時間を表示
* 入力したデータを LocalStorage に保存
* 当日を枠線で表示
* 当日を含む過去7日間の合計が28時間を超えた場合、日付の背景を赤く表示
* 前月に入力した勤務時間も7日間の計算に使用

## How to Use

1. 年月を `YYYYMM` 形式で入力します
   例: `202605`

2. `表示` ボタンを押します

3. 各日付の入力欄に勤務時間を入力します

4. 入力した勤務時間はブラウザに自動保存されます

5. 過去7日間の合計が28時間を超えると、その日付が赤く表示されます

## Notes

勤務時間のデータはブラウザの LocalStorage に保存されます。

別のブラウザや別の端末では、保存したデータは共有されません。
