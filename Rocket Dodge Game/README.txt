# Rocket Dodge Game

JavaScriptとCanvasを使用して作成したシンプルな回避ゲームです。

ロケットを矢印キーで操作し、上から落ちてくる星を避けます。
星に10回当たるとゲームオーバーになります。

## 遊び方

1. `Start` ボタンを押してゲームを開始します
2. 矢印キーでロケットを操作します
3. 上から落ちてくる星を避けます
4. 星に当たると `Hits` が増えます
5. `Hits` が10になるとゲームオーバーです
6. `Restart` ボタンで最初の画面に戻ります

## 操作方法

* `←`：左へ移動
* `→`：右へ移動
* `↑`：上へ移動
* `↓`：下へ移動

## 機能

* Canvasを使用したゲーム画面
* 矢印キーによるロケット操作
* ランダムな位置と速度で星を生成
* ロケットが画面の外に出ないように制御
* ロケットと星の当たり判定
* スコア表示
* ヒット数表示
* ゲームオーバー表示
* Start機能
* Restart機能

## 使用技術
