# Weather App

Open-Meteo APIを使用して作成したシンプルな天気アプリです。

都市名を入力すると、現在の気温、体感温度、湿度、風速、天気の状態を表示します。

## 使い方

1. 入力欄に都市名を入力します
   例: `Kyoto`、`Tokyo`、`London`

2. `検索` ボタンを押します

3. 検索した都市の現在の天気情報が表示されます

## 機能

* 都市名による検索
* 都市の緯度と経度を取得
* 現在の気温を表示
* 体感温度を表示
* 湿度を表示
* 風速を表示
* 天気の状態を日本語で表示
* 天気に応じてアイコンを変更
* 天気に応じてカードの背景色を変更
* 都市が見つからない場合のエラー表示
* API通信に失敗した場合のエラー表示

## 天気の表示

天気コードに応じて、以下の状態を表示します。

* 晴れ
* 曇り
* 霧
* 雨
* 雪
* 雷雨

## 使用技術

* HTML
* CSS
* JavaScript
* Fetch API
* Open-Meteo API

## API

このアプリでは、以下のOpen-Meteo APIを使用しています。

* Geocoding API：都市名から緯度と経度を取得
* Weather Forecast API：現在の天気情報を取得
